本项目包括 登录，增删改查，文件上传，权限检测，图像生成，收纳式展示，图表展示，爬虫获取数据并写入excel，模糊搜索等。

食用方式：开袋即食 ！
首先安装所需环境对应上方各版本的软件，Windows,Linux均有对应。
